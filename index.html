Introduction

making a website is a very simple and easy proccess. The First steps is to lay out the basics of the website like the doctype and the head and body.

Here is a list of some basic elements to start with.

  1. !DOCTYPE html with brackets.
  2. head wich includes the title of the website and the link to your css.
  3. The make the body wich will include the majority of your website like text, images, videos, colors, fonts, etc.
  4. Also Dont forget the opening and closing brackets of what the text is going to be like a p for paragraph.

List

List are a very useful structure for a website and help organize things.

  1. Use ol for ordered lists.
  2. Use ul for unordered lists.
  3. inside either ol or ul should be li for list.

How to Upload your website to Github!

Uploading to git hub is very usfull for collaberating with friend and colleges on a coding project.

Steps to upload to GitHub.

  1. Search Git Bash on google and download the software.
  2. Once the software is downloaded go to the folder were your code is located and right click inside the folder and click git bash.
  3. A comand line will pop up "don't be afraid it is fairly easy. Inside the command line type "git init ".
  4. After that type "git checkout -b gh-pages".
  5. After that type "git status".
  6. You will then see all the files in the follder in red wich means the files are not added.
  7. To add the files type "git add "FILENAME HERE" do this for all the red files listed.
  8. After that type "git status" again to see if it worked.
  9. If you add things to your code your are going to want to type "git commit -m "MESSEGE"" that way everything is updated.
  10. Then type "git status" and everything should be white.
